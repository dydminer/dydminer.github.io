矿机护卫-看门狗

发表于 2018-02-03 | 分类于 挖矿工具 | | 阅读次数

前言

没有人能够保证自己的矿机能够一直不会死机，更尴尬的是死机的时候你正在公司上班或者陪领导钓鱼，此时你怎么办？等着晚上回家再重启？你就损失了好几个甚至几天的收益，本篇我推荐大家一个挖矿神器-看门狗

开源地址：https://github.com/dydminer
个人博客：http://dyd.cppba.com
eth矿工交流群：175023845

使用场景

我们在鱼池挖矿，突然有一天一台矿机死掉了，这个时候10分钟这个矿机没有提交算力，鱼池就会发送微信警告。我们就可以定位到是哪一台矿机失联，这时我们只需要拿出手机打开看门狗APP，远程强制关机，重启就OK！！！

产品推荐

淘宝上同类产品很多，大家可以通过关键字“USB看门狗”找到，价格最便宜的只需要20元左右，当然也有稍微贵一点的。

一般情况下二十几元的就能满足绝大部分的需求，如果你不放心你也可以像博主一样买一款稍微贵一点的。

俗话说一分钱一分货，博主买的是一款是由“凡达电子”生产，价格在59.8元，他的优点是可以使用手机APP远程关机和启动（这里就不附购买链接了，有兴趣可以加我私人QQ或者交流群探讨）

收到货以后，你会发现看门狗外观还是非常简陋，没有做过多包装，所以要格外小心，千万别碰断了阵脚或者弄掉一个零件，

我们只需要包看门狗的两个power针脚插到主板的开关机阵脚上（这里有的朋友会担心以前的开机按钮插哪里，老板会送一根分叉线，分出来可以一根继续接以前的开机按钮，另一根接看门狗），

配置也很简单，卖家会给你一个说明书，照着说明书步骤做旧行了，支持Android和iOS的app，你照着配置就可以了，非常简单，配置好以后你会在APP上看到这样的界面

“rx588001”是博主的其中一台矿机，我们看到上面有一个长的像电灯开关的东西“OFF”，这个就是远程操控矿机电源开关的按钮，平时注意别误碰到了，尽量把APP藏得隐秘一点，特别是有小孩的家长
